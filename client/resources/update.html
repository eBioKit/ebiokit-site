How to update your eBioKit

Step 1. Upgrade application code

Go the the location of the eBioKit code and pull the latest version from GitHub.

cd /your/ebiokit/app/location/

git pull origin minified

Step 2. Install missing python packages (if any)

Activate virtualenv (if using) and install the new Python packages using pip.

source .env/bin/activate

pip install -r requirements.txt

Step 3. Finally, upgrade the databases

Run the following command and Django will automatically upgrade the database.

cd server

python manage.py makemigrations

python manage.py migrate

Done! Your system should be now upgraded to latest version!

Enjoy the eBioKit and please contact us for any question or suggestion!
